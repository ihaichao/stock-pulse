# Stock Pulse

> 📡 股票事件驱动提醒系统 — 让你不再错过影响持仓的关键事件。

Stock Pulse 是一个面向美股散户的事件日历 + 智能提醒网站。用户输入自己的持仓列表，系统自动聚合未来和过去与持仓相关的关键事件（财报、宏观数据、内部人交易、分析师评级等），并用 AI 生成简明解读。

**核心理念：** 不告诉你买什么卖什么，只确保你「不会在不知情的情况下被事件打个措手不及」。

---

## 产品定位

### 解决什么问题？

普通散户投资美股时，最容易踩的坑之一是：

- 不知道今晚自己持仓的某只股票要发财报
- 不知道今天有 FOMC 会议、CPI 数据发布
- 不知道某个高管刚刚大量减持了自己持有的股票
- 事后才发现「原来昨晚发生了这么多事」

现有券商虽然有日历功能，但：

- 信息散落在各处，不与你的持仓关联
- 没有「人话解读」，只有冷冰冰的数据行
- 宏观事件和个股事件是割裂的

### Stock Pulse 的差异化

1. **以持仓为中心**：所有信息围绕「你持有什么」来组织
2. **事件驱动**：不是行情软件，是「事件日历 + 提醒」
3. **AI 解读**：每个事件都有一段简明中文解释，帮你快速判断重要性
4. **SEO 友好**：个股事件页可以被搜索引擎收录，带来自然流量

## 核心页面设计

### 1. 首页 / 仪表盘（登录后）

**布局：**

- **顶部时间线**：未来 7 天与用户持仓相关的事件
  - 今天的事件高亮
  - 每个事件一行：`股票代码 | 事件类型图标 | 简短描述 | 重要性标签`
  - 重要性分三档：🔴 高（财报/FOMC）、🟡 中（分析师评级变动）、🟢 低（常规宏观数据）
- **下方卡片区**：
  - 「今日关注」摘要卡片（最重要的 3–5 条事件）
  - 「昨夜回顾」摘要卡片（昨晚已发生的事件 + 结果概要）

### 2. 今日事件页

**两个 Tab：**

- **今日预览**：今晚美股开盘后将要发生的事件
  - 按重要性排序
  - 每条事件：代码、类型、时间（盘前/盘中/盘后）、简短描述
- **昨夜复盘**：昨晚已经发生的事件及结果
  - 财报：实际 EPS vs 预期、营收 vs 预期、盘后涨跌幅
  - 宏观数据：实际值 vs 预期值、市场反应
  - 内部人交易：谁买/卖了多少

### 3. 个股事件页（SEO 核心页面）

**路由：** `/stock/:ticker`（如 `/stock/NVDA`）

**内容：**

- 股票基本信息（公司名、行业、当前价格）
- 事件时间线（未来 + 过去）：
  - 财报历史（过去 4–8 个季度）
  - 内部人交易记录
  - 分析师评级变动
  - 关键新闻事件
- 每个事件可展开查看 AI 解读

**SEO 价值：**
- 用户搜索「NVDA 财报日期」「AAPL earnings date」等长尾词时，可能直接命中此页面
- 页面标题/描述模板化，便于搜索引擎抓取

### 4. 宏观日历页

**路由：** `/macro`

**内容：**

- 月度日历视图
- 标注所有重要宏观事件：
  - FOMC 利率决议
  - CPI / PPI
  - 非农就业数据
  - GDP
  - 初请失业金
  - 其他（PMI、消费者信心指数等）
- 每个事件标注：预期值（consensus）、前值、实际值（已公布时）
- 点击可看 AI 简要解读

**高频刚需：** 很多投资者每周都会查「这周有什么重要数据」，这个页面可以成为用户每周必访入口。

### 5. 事件详情页

**路由：** `/event/:id`

**内容（根据事件类型动态展示）：**

- **财报事件：**
  - EPS / 营收：实际 vs 预期 vs 前值
  - 管理层指引要点（AI 总结）
  - 盘后/盘前股价反应
  - 历史财报表现趋势（过去几个季度是 beat 还是 miss）

- **内部人交易（Form 4）：**
  - 谁买/卖了多少股、均价
  - 是计划内（10b5-1）还是计划外
  - 这个人的历史交易记录
  - 同行业近期内部人交易趋势

- **宏观数据：**
  - 实际值 vs 预期 vs 前值
  - AI 解读：这个数据对市场的可能影响
  - 历史数据趋势图（简单折线）

- **分析师评级变动：**
  - 哪家机构、从什么评级变到什么评级
  - 目标价变动
  - 该机构历史评级准确度（如果数据可获取）

## MVP 范围（第一版）

> 原则：先做能跑通的最小闭环，快速验证用户是否真的会用。

### MVP 只做三件事：

1. **持仓管理**
   - 用户输入自己的股票列表（手动添加/删除 ticker）
   - 暂不做真实券商账户连接
   - 数据存储：先用简单的用户标识 + 服务端存储（或 localStorage）

2. **事件日历**
   - 展示未来 7 天跟持仓相关的事件
   - **MVP 只做两类事件：**
     - 📊 Earnings（财报发布日期 + 预期 EPS/营收）
     - 📅 宏观经济数据（FOMC、CPI、非农等主要事件）
   - 数据来源：公开 API / 免费数据源

3. **每日摘要页**
   - 一个「今天你需要关注什么」的页面
   - 列出今天/今晚与你持仓相关的事件
   - 每条事件附一段 AI 生成的简短解读（1–3 句话）

### MVP 不做的（Phase 2+）：

- 用户注册/登录系统（MVP 可以用简单 token 或 localStorage）
- 内部人交易、分析师评级（数据获取成本较高）
- 个股事件页的完整历史时间线
- 邮件/Telegram 推送提醒
- SEO 优化（先把功能做对，再做流量）

Stock Pulse 提供的是**事件信息聚合与教育辅助工具**，不构成任何形式的投资建议。
